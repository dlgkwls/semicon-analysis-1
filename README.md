## The Data Sourse
I cleaned a csv file named chip_companies_financials to create the data I used which was from Kaggle Datasets Global Semiconductor Industry 2010-2026. Link: [https://www.kaggle.com/datasets/sergionefedov/global-semiconductor-industry-2010-2026]

## I have learned
* How to clean a raw csv file using basic python skills and pandas libarary (for loop, read_csv, shape, isnull, sum, strip, etc)
  And how to shape the data that it's easier to form plots
* How to diversify the charts using matplotlib.pyplot libarary (figure, xlable, title, etc)
* How to create various chart types (line, bar, scatter plots)

## I have found
The three charts I created indicates:
1. Nvidia started to exponentially grow from 2022
2. idm_memory earned the most revenue out of other segments in semiconductor industry in 2024
3. fabless_gpu segment got the most revenue in contrast of their R&D funding in 2024
